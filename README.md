**1. Hanti-dhawr — ERP for Somaliland SMEs**

- **Backend Skill Signal:** Large-scale relational schema design, authentication architecture, role-based access control, dashboard-connected analytics

- **Backend Skill Signal:** End-to-end full-stack integration, database connection troubleshooting and configuration, practical debugging under real environment constraints
- **Portfolio Value:** Shows the ability to take a full-stack system from setup issues to a working, connected product
